<h2 align="center">开始gitPro之旅</h2>

gitPro是一个基于微信小程序的自制项目,可以在项目上浏览查看各种github项目,并根据github API V3接口进行基础的订阅等基础操作, 如果你对这个项目有兴趣可以帮忙修改项目,虽然项目有很多不足之处,但是效果初衷是好的也是我第一次创建微信小程序项目
项目效果类似于GitHub Pro的效果，但是使用的技术会持续更新, 希望大家喜欢。

## 介绍
gitPro是一个基于微信小程序的前后端分离项目, 使用了github API V3, 并可以连接本地服务器, 本地数据库进行数据处理。

#### 浏览器支持  

由于electron是基于node.js和chromium内核的项目,所以iBEV浏览器模拟采用chrome及其内部插件扩展。

## 安装

1.	将项目clone后导入微信小程序开发者工具内
2.	如果需要本地数据处理, 请修改server.js服务器文件
3.	如果你需要对后续项目进行开发,请确保安装了node.js 

```sh
# npm install node
```

#### 平台支持
支持不同机型使用, 可以自动适配手机屏幕。
